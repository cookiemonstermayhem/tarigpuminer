# xtrgpuminer


Set `config.json` file as such:

```
{
  "tari_address": "<address>",
  "tari_node_url": "http://127.0.0.1:18182",
  "coinbase_extra": "tarigpu",
  "template_refresh_secs": 30
}
```

Use `tari_node_url` to minotari_node grpc.
